# Treasury Prototype (Frontend Only)

## Overview
This is a simple frontend-only prototype built for the Treasury take-home assessment. It demonstrates basic UI interaction and state handling without a backend.

## Features
- Add items to a list
- Simple responsive layout
- Basic DOM interaction

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
Just open `index.html` in a browser.

OR use Live Server in VS Code.

## Deployment
This project can be deployed using:
- Netlify
- Vercel
- GitHub Pages

## Approach
Kept intentionally simple to ensure fast deployment and easy testing.

## Assumptions
- No backend required
- No database required
- Runs fully in browser
